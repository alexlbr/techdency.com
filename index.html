ReactJS training requirements


In order to participate in the ReactJS one-day intensive training you need to have a good understanding of Javascript, ES6, and testing.

An interview has to take place to attend the training. Attendees need to have a minimum and similar skill level so we can maximise the learning.

If you do not have yet the skills to participate in the React training you can apply to the two-day Pre React training (more info below) which gives you access to the one-day ReactJS training session. Notice an interview is also required to participte in the Pre-React training.

Pre-ReactJS Training

Master JavaScript in 2 intensive days. 10 developers max.

Curriculum

Day 1 (Mastering Javascript fundamentals)

  • Getting started with node, npm, git and Github

Day 2 (Advanced Javascript)
